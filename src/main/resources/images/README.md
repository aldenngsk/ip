# Avatar images

Put your avatar images here to replace the grey placeholders in the chat:

| File name   | Used for              | Suggested size |
|------------|------------------------|----------------|
| `user.png` | Your messages (right) | 40×40 to 80×80 px |
| `duke.png` | SixSeven’s messages (left) | 40×40 to 80×80 px |

Supported formats: PNG, JPG, GIF.

If a file is missing, the grey circle placeholder is shown instead.
